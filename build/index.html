Client/Server in C

This client/server system implements a "string modification" protocol using TCP. The client sends an opcode and payload to the server in a single formatted packet. The opcode describes how the server should modify the string. The server modifies the string, packages it in a packet, and sends it back to the client. The client then displays the payload for the user.

Tokenizer in C

I wrote this tokenizer as part of an automata theory/compiler construction class. The program takes as input text files that contain programs written in a simplified Pascal-like language. It reads each line of the program and breaks the line up into individual tokens and labels the tokens based on if it is a keyword, variable type, variable, or some symbol that is valid in the Pascal-like language. Now these tokens are ready to be passed to a parser.
